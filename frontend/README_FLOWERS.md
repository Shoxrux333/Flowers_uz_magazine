# Flowers UZ - Frontend

Ushbu loyiha Flowers UZ veb-sahifasining frontend qismi hisoblanadi. React va Tailwind CSS texnologiyalari asosida qurilgan.

## Xususiyatlari
- **React 18**: Zamonaviy React komponentlari va hooks.
- **Tailwind CSS**: Tezkor va oson CSS framework.
- **Lucide React**: Chiroyli va shuq ikonlar.
- **JWT Authentication**: Xavfsiz foydalanuvchi autentifikatsiyasi.
- **Shopping Cart**: Mahsulotlarni savat qilish va to'lov qilish.
- **Responsive Design**: Mobil, planshet va desktop uchun optimallashtirilgan.

## O'rnatish va ishga tushirish

1. **Kutubxonalarni o'rnatish**:
   ```bash
   npm install
   ```

2. **Development serverini ishga tushirish**:
   ```bash
   npm start
   ```

3. **Production uchun build qilish**:
   ```bash
   npm run build
   ```

## Konfiguratsiya

Backend URL'ni o'zgartirish uchun `src/App.jsx` faylida `API_BASE` o'zgaruvchisini o'zgartiring.
